# 页面编辑说明

欢迎参与本项目的文档编写！

我们将在本页面中指出对于页面修改的一些注意事项和规范，帮助你更好地进行内容编辑和维护

## 1. 基础语法支持

本项目支持所有标准的 Markdown 语法，您可以在这里找到所有的基础语法教学：
- [Markdown 基础语法](https://markdown.com.cn/basic-syntax/)

---

## 2. 目录结构与内容修改
为了保持文档的清晰和易维护，所有页面按照如下规则进行读取

* 文件夹名称即为页面中`目录`的显示名称，以`.`开头命名的所有文件和目录不会显示在实际页面中
* 建议将图片存放在`images`文件夹中，或者任意一个以`.`开头的目录中，避免造成读取混乱
* 所有需要读取的文件均为`.md`格式
* 提交内容时，请先提交`Pull Request`，并在 PR 描述中注明修改内容和目的，等待审核后合并

---
## 3. 页面中的自定义内容

为了让文档更加生动、重点更加突出，我们在前端渲染时引入了 **7 种不同颜色** 的自定义提示容器

感谢Napcat文档的自定义容器设计，请在Github上支持他们 [Napcat文档](https://github.com/NapNeko/NapCatDocs)

你只需要使用 `:::` 包裹内容，并在第一行写上容器的类型即可。如果需要**自定义标题**，直接写在类型后面加个空格即可。

### 📌 Info (常规信息)

最基础的灰白色提示框，适合补充说明辅助信息。

```markdown
::: info
这里是一些补充的背景信息。
:::
```

** 实际渲染效果：**

::: info
这里是一些补充的背景信息。
:::

### 🟩 Tip (推荐/提示)

绿色的提示框，适合分享最佳实践或推荐做法。

```markdown
::: tip
推荐使用 Vite 构建你的前端项目，速度会快很多。
:::
```

** 实际渲染效果：**

::: tip
推荐使用 Vite 构建你的前端项目，速度会快很多。
:::


### 🟨 Warning (警告)

黄色的警告框，适合提醒读者容易犯错的地方。

```markdown
::: warning 核心注意点
在执行此命令前，请确保你已经在正确的目录路径下。
:::
```

** 实际渲染效果：**

::: warning 核心注意点
在执行此命令前，请确保你已经在正确的目录路径下。
:::


### 🟥 Danger (危险)

红色的危险提示框，用于极其危险的操作，防止读者产生不可逆的损失。

```markdown
::: danger ⚠️ 严禁操作
绝对不要在生产环境运行 `rm -rf /`，这会导致数据永久丢失！
:::
```

** 实际渲染效果：**

::: danger ⚠️ 严禁操作
绝对不要在生产环境运行 `rm -rf /`，这会导致数据永久丢失！
:::


### 🟦 Note (笔记/记录)

蓝色的笔记框，适合作为额外扩展的阅读资料。

```markdown
::: note 历史记录
在 V2.0 版本之前，这个 API 的参数名称是不同的。
:::
```

** 实际渲染效果：**

::: note 历史记录
在 V2.0 版本之前，这个 API 的参数名称是不同的。
:::


### 🟪 Important (重要考点)

紫色的强调框，用于全文最核心、要求必须掌握的重点内容。

```markdown
::: important 必考知识点
无论你使用哪种框架，组件的生命周期都是必须深入理解的概念。
:::
```

** 实际渲染效果：**

::: important 必考知识点
无论你使用哪种框架，组件的生命周期都是必须深入理解的概念。
:::


### 🌺 Caution (谨慎处理)

淡红色的谨慎提示框，级别介于 Warning 和 Danger 之间。

```markdown
::: caution 谨慎升级
此版本包含了破坏性更新（Breaking Changes），升级前请仔细阅读更新日志。
:::
```

** 实际渲染效果：**

::: caution 谨慎升级
此版本包含了破坏性更新（Breaking Changes），升级前请仔细阅读更新日志。
:::

## 4. 代码块展示

支持标准的语法高亮格式（在代码块开头标明语言名称）：

``` java
// 这是一个 Java 代码块示例
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
